3D프린터 목업 제작 지원

3D 프린터 목업이란?
  • 단말사의 개발을 위해 필요한 경우 최대 2개까지 3D 프린트를 이용한 목업 샘플 제작을 지원합니다.
3D 데이터 제작시 유의사항
  • 출력 가능한 3D 데이터 종류(확장자)는 PRO, STL, CTL, OBJ, PLY, ZPR, ZBD, AMF, WRL, 3DS, FBX, IGES, IGS, STEP, STP, MJPDDD 입니다.
  • 제작 신청시 데이터 두께가 정의된 Solid Type 이어야 합니다. (Surface Type 으로는 출력 불가)
  • 조립이 필요한 모델인 경우 결합 구조를 고려하여 부품을 분리 하여 표기해주셔야 합니다.
  • 작성된 3D 데이터의 각 부품이 3D 프린터 최대 출력 사이즈를 초과할 수 없으며 최대 출력 사이즈는 (W x L x H): 380mm x 380mm x 250mm 입니다.
3D 프린터 제작요청
  • 3D 프린터 목업 제작 신청을 원하시면 아래 신청서류를 다운받아 작성 후 “KT IoT Dev → 프로세스 → 개발지원 → 3D목업제작지원” 에서 신청하시면 됩니다.
  • 신청 서류는 “KT IoT Dev → 프로세스 → 개발지원 → 3D목업제작지원” 에서도 다운받으실 수 있습니다.

*신청서류는 PC웹 KT IoT Dev에서만 다운로드 가능합니다.
