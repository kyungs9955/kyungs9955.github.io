모듈/모뎀

PM-L410K
종류 모듈
통신망 LTE-M
모뎀 제조사 우리넷
칩셋 제조사 퀄컴
칩셋명 MDM9207
Category Cat.1
제조사 홈페이지 http://www.woori-net.com
Description

- 단말기는 농기계(이앙기,트랙터)에 장착되어 RTK-GPS 기술 기반 실시간 정밀측위 목적으로 사용됨.
- 정밀측위 기능 수행을 위해 단말기는 KT 보정 데이터 생성 서버로부터 가변 크기의 보정 데이터를 주기적으로 수신함.
- 단말기로부터 측정된 위치 데이터는 CAN 인터페이스를 통해 농기계 CAN 버스에 주기적으로 전송됨.

Software
단말 OS REX / LINUX
SMS 지원 Type KSC5601/UCS2
WCDMA BAND 해당 없음
지원 BAND LTE(BAND 3 & BAND 8)
칩셋명 MDM9207
USIM 타입 Micro
3GPP Release 10.0.0
Hardware
기기타입 모뎀
모뎀 유형 모듈 내장형
모뎀 사이즈 (W*D*H mm) 25*34*15
무게 (g) 250
메모리 사양 NAND 1Gb, DDR 512Mb, EFS Size 512Mb
CPI 사양 ARM926EJS-230MHz
전원 사양 DC 12V
제공 인터페이스 Micro USB, UART, LED Indicators
Package type Package type A
회사소개 이용약관 법적고지 개인정보처리방침 개인정보 이용내역 청소년보호정책
