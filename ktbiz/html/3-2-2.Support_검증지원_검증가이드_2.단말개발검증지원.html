2. 단말/개발/검증 지원
2.1 kt Device 고객지원 사이트
2.1.1 kt Device 고객지원 사이트 가입
  1. ① https://iotdev.kt.com (크롬으로 접속 권장) → PSSO 회원 가입
  2. ② 사용자 등록 선택 → 승인된 PSSO ID/PW 로그인 → 회사 등록 → 사용자 등록 (1회사당 1개 ID만 생성)
  3. ③ 정보보호 서약서 ( 제조사 대표, 담당 개발 직원 작성 및 스캔 후 한 개의 파일로 등록)
    • [사용자 등록] 내 [양식다운]을 통해 정보보호 서약서 양식을 다운로드 가능
    • kt 관계부서는 “단말품질관리팀”으로 기재
2.1.2 단말 검증 요청 및 업무 지원
  1. 1) 단말 검증 요청
    1. ① 메뉴 : 검증 → 개발지원 → 모델 등록(단말 등록) → QAT - 검증 요청
    2. ② 내용 : 검증대상 모델 등록, QAT / 기능 검증 요청, 검증결과 확인, 제조사 답변 등의 기능 제공
  2. 2) 기타 업무지원
    1. ① 개발지원 : 사전 회의, 결함 재연, 개발 중 문의사항 등의 요청/문의 기능 제공
    2. ② 기술지원 : 상용화 진행 후 문의사항, VOC 확인 요청, 기술적 문의사항 등의 기능 제공
    3. ③ 자료 다운로드 : 규격서, 가이드, 입고문서 등의 다운로드 기능 제공
