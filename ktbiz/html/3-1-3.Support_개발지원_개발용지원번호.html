개발용 번호 지원

개발용 지원 번호란?
  • 단말사의 개발을 위해 eMTC 시험용 번호를 최대 2회선까지 유심을 포함하여 지원합니다.
  • IoT 번호는 번호이동성이 지원되지 않기 때문에 개통한 단말 외 다른 단말에 꽂아서 사용하실 수 없습니다.
  • 시험용 번호는 시험용 외에 다른 용도로 사용할 수 없으며 일정기간이 지나면 기간만료가 되어 사용 불가능 할 수 있습니다.
개발용 번호 지원 신청
  • 시험용 번호 신청을 원하시면 아래 신청서류를 다운받아 작성 후 “KT IoT Dev → 프로세스 → 개발지원 → 시험번호신청” 에서 신청하시면 됩니다.
  • 신청 서류는 “KT IoT Dev → 프로세스 → 개발지원 → 시험번호신청” 에서도 다운받으실 수 있습니다.
