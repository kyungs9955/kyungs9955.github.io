검증 컨설팅

검증 프로세스 컨설팅
  • 전체 상용화 프로세스 안내
  • 개발 상황 확인을 통한 맞춤형 진행 프로세스 안내
  • 개발완료 → QAT/기능검증 → 초도 물품검수 → 단말출시 부분별 프로세스 안내
  • 개발 상황 확인을 통한 맞춤형 진행 프로세스 안내
  • SAT 진행 프로세스 안내
필수적용 기능 구현 컨설팅
  • 상용화 이후 원격 지원기능(FOTA, 품질정보, 원격 RESET 등) 구현여부 확인 및 컨설팅 진행
  • 검증 환경 요구사항(CM/DM 구성, End To End 등) 확인 및 컨설팅 진행
  • 서비스 단말의 보안, 안정성, 망 영향 등을 고려한 전문적인 컨설팅 진행
진행일정 컨설팅
  • 상용화 일정에 맞추기 위한 효율적인 검증 스케줄 컨설팅 진행
  • 수정 개발 목표 컨설팅 진행
  • 검증 프로세스 별 개발사 대응 일정 컨설팅 진행

KT QAT 컨설팅은 KT사업부서를 통한 단말품질관리팀으로의 요청이 있을 경우 KT IoT Dev사이트를 통해 지원하고 있습니다. 검증 준비 시간 단축을 위해 KT IoT Dev사이트를 통해 컨설팅 요청을 주시기 바랍니다.
